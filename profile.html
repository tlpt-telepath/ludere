自己紹介

tlpt はフロントエンドとビジュアルデザインを起点に、ストーリーのあるインタラクションづくりを得意としています。ユーザーリサーチを基にしたワイヤリングから、WebGL やモーションを組み合わせた実装まで一貫して担当。「思わず触りたくなる UI」を探求しています。

スキルセット

  • デザインシステム設計と UI デザイン
  • TypeScript / React / Next.js を用いたフロントエンド開発
  • Three.js / WebGL を活用したインタラクション制作
  • プロトタイピングとユーザーテスト設計

主な制作

  • 2024

    Immersive Brand Promo

    ブランドサイトのリニューアルで WebGL 演出と音響反応を実装。離脱率を 28% 改善し、キャンペーン期間のPVを過去最高に更新。

  • 2023

    Playful Learning Prototype

    ludere 内の R&D プロジェクトとして、日常的に使える学習導線をモーションで補強。ユーザーテストで「継続したくなる体験」と評価。

  • 2022

    Festival Explore App

    都市型フェス向けナビゲーションアプリを共同制作。GPS を活用したインタラクティブマップと行動ログの可視化を担当。

ステートメント

「遊びの余白があるほど、人は主体的に物語を紡げる」。tlpt は ludere のメンバーとして、クライアントとユーザーの間にある体験を拡張することを使命としています。
